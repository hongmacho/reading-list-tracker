# UI/UX 가이드: BookMark

## 1. 디자인 철학

**BookMark**는 **극단적 단순함**을 지향합니다.
- 독서라는 주요 활동에 방해가 되지 않도록 최소한의 UI만 제공
- 따뜻하고 편안한 톤으로 독서 경험을 보조
- 모바일 우선 설계로 이동 중 독서 기록 가능
- 어두운 환경(밤 독서)에 배려한 다크 모드 지원

## 5. 레이아웃 구조

### 전체 애플리케이션 레이아웃

```
┌─────────────────────────────────────┐
│    Header / Navbar (고정)            │
│  [📚 BookMark] [검색] [프로필]      │
├─────────┬───────────────────────────┤
│         │                           │
│ Sidebar │      Main Content        │
│ (접고    │      (반응형)            │
│  펼침)   │                           │
│         │                           │
│  • 대시보드 │                           │
│  • 서재   │                           │
│  • 통계   │                           │
│  • 설정   │                           │
│         │                           │
└─────────┴───────────────────────────┘
```

### 반응형 레이아웃 (Mobile First)

| Breakpoint | 화면폭 | 특성 |
|-----------|--------|------|
| **xs** | 320px - 640px | 모바일 (아이폰 SE ~ 12) |
| **sm** | 640px - 768px | 소형 태블릿 |
| **md** | 768px - 1024px | 태블릿 (아이패드) |
| **lg** | 1024px - 1280px | 노트북 |
| **xl** | 1280px+ | 데스크톱 모니터 |

**Tailwind Breakpoint 적용**:
```tsx
// 모바일: 768px 이하에서는 사이드바 숨김
<aside className="hidden md:block w-64 bg-brown-50">
  {/* Sidebar */}
</aside>

// 콘텐츠: 반응형 패딩/마진
<main className="p-4 md:p-6 lg:p-8">
  {/* Main */}
</main>
```

### 스페이싱 시스템 (Tailwind Spacing)
```
4px  = 1 unit  (p-1)
8px  = 2 units (p-2)
16px = 4 units (p-4)
24px = 6 units (p-6)
32px = 8 units (p-8)
```


## 8. 상호작용 & 애니메이션

### 버튼 상태 (Button States)

| 상태 | 스타일 | 커서 |
|------|--------|------|
| **Default** | 갈색 배경, 흰색 텍스트 | pointer |
| **Hover** | 더 진한 갈색 | pointer |
| **Active/Pressed** | 배경 어두워짐, 작아짐 | pointer |
| **Disabled** | 회색 배경, 흐린 텍스트 | not-allowed |
| **Loading** | 로딩 스피너 표시 | wait |

**구현** (Tailwind + Framer Motion):
```tsx
<button className="
  bg-brown-600 hover:bg-brown-700 active:scale-95
  text-white px-4 py-2 rounded-lg
  transition-all duration-200
  disabled:bg-gray-400 disabled:cursor-not-allowed
">
  저장
</button>
```

### 페이지 전환 애니메이션
- **Fade In**: 페이지 로드 시 0.3s fade-in
- **Slide In**: 모달/드로어 0.2s slide-in-right

### 호버 효과
- **카드 호버**: 그림자 강화 + 약간 위로 올라옴 (lift)
- **링크 호버**: 밑줄 나타남 + 색상 변경

## 10. 접근성 (a11y)

### 키보드 네비게이션
- **Tab**: 다음 요소로 이동
- **Shift+Tab**: 이전 요소로 이동
- **Enter**: 버튼/링크 활성화
- **Space**: 체크박스/라디오 선택
- **Escape**: 모달 닫기
- **Ctrl+K**: 검색 열기 (glo bal shortcut)

### ARIA 레이블

```html
<button aria-label="새로운 책을 서재에 추가합니다">
  책 추가
</button>

<input aria-label="제목으로 검색" placeholder="제목, 저자, 인용" />

<div role="alert" aria-live="polite">
  저장되었습니다.
</div>
```

### 색상 대비
- **WCAG AA**: 최소 4.5:1 (텍스트)
- **WCAG AAA**: 최소 7:1 (텍스트)

모든 텍스트가 배경에 대해 충분한 대비를 가져야 함.

### 이미지 alt 텍스트
```tsx
<img 
  src="/book-cover.jpg" 
  alt="'클린 코드' 책 표지, 저자 로버트 C. 마틴"
/>
```
